# 🐦 Flappy Bird - C# Windows Forms (ENG - TUR)

This project is a simple and fun clone of the classic Flappy Bird game, developed using C# and Windows Forms.

## 🎮 Features
* Continuous pipe movement and gravity mechanics.
* Collision detection (Game over if the bird hits the pipes or the ground).
* Real-time score tracking.
* Smooth jumping controls.
